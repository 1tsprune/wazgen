RULE_DEPLOY



  
Deploy Rule ke Server

  

    Setelah rule di-generate, simpan sebagai local_rules.xml dan jalankan perintah berikut:
  

  

    
    
sudo cp local_rules.xml /var/ossec/etc/rules/
sudo chown root:wazuh /var/ossec/etc/rules/local_rules.xml
sudo chmod 640 /var/ossec/etc/rules/local_rules.xml
echo 'your log here' | /var/ossec/bin/wazuh-logtest
sudo systemctl restart wazuh-manager

Smart Parser

    
  

  

    
Untuk log yang tidak ada di database, smart parser akan auto-detect format dan generate rule sesuai:

    
    
      
  • JSON — auto-parse all fields
    • 
      
  • Syslog — timestamp + program + message
    • 
      
  • Key=Value — extract key-value pairs
    • 
      
  • Windows Event — EventID + fields
    • 
      
  • HTTP Access — Apache/Nginx format
    • 
      
  • Generic — keyword + IP detection
